AI idea

CareBot answers common support questions.

CareBot uses Groq through a Vercel serverless proxy, so the browser never receives the secret API key. It is designed for FAQs, support navigation, and compassionate non-medical guidance.

Safe scope: CareBot does not diagnose or replace doctors. For emergencies, it advises contacting local emergency services or a hospital immediately.
